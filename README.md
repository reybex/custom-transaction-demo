# Reybex Example Transaction

Bu proje, Reybex frontend uygulaması içinde iframe olarak gösterilebilecek örnek bir React uygulamasıdır.

## Özellikler

- React + TypeScript + Vite
- Tailwind CSS ile modern UI
- Ürün kartları gösterimi
- Reybex API entegrasyonu
- iframe içinde token yönetimi

## Kurulum

```bash
cd ReybexExampleTransaction
npm install
# veya
yarn install
```

## Çalıştırma

```bash
npm run dev
# veya
yarn dev
```

Uygulama `http://localhost:3000` adresinde çalışacaktır.

## Kullanım

Bu uygulama, Reybex frontend uygulaması içinde iframe olarak açılmalıdır. Ana uygulamadan token alarak API çağrıları yapar.

## Build

```bash
npm run build
# veya
yarn build
```
